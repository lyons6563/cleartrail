Demo Case 001 — Late and Missing 401(k) Contributions

Scenario:
A mid-size employer processes bi-weekly payroll. Two payrolls in Q1 had issues:
• One payroll was deposited late.
• One payroll was partially deposited.

Inputs:
• payroll.csv — employee deferrals by pay date
• trust.csv — bank deposit activity
• recordkeeper.csv — participant postings (optional)

Expected outcomes:
• One late deposit flagged with days-to-deposit
• One missing/partial deposit flagged with variance
• Exception log generated
• Timeliness report generated
• Evidence index created

This demo shows how the engine replaces manual Excel reconciliation and produces audit-ready outputs.
