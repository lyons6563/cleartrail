# Appendix

## Purpose
This folder contains supporting documentation, source data files, communications, and other evidence referenced in the main reconstruction documents.

---

## Folder Structure

### Data/
Contains source data files, exports, and extracts used in the reconstruction.

### Financial/
Contains bank statements, wire transfer confirmations, deposit records, and other financial documentation.

### Legal/
Contains plan documents, amendments, legal opinions, and regulatory guidance.

### Communications/
Contains emails, meeting notes, and other correspondence related to the case.

### Reports/
Contains third-party reports, reconciliation reports, and analysis documents.

### Systems/
Contains system documentation, configuration files, and technical specifications.

### Statements/
Contains participant statements, account statements, and other participant-facing documents.

---

## File Naming Convention

Files should be named using the following convention:
`[Category]_[Description]_[Date].[ext]`

Examples:
- `Data_Payroll_Export_Q1_2024.csv`
- `Financial_Bank_Statement_Jan_2024.pdf`
- `Legal_Plan_Amendment_2023-12.pdf`
- `Communications_Email_Deposit_2024-01-08.pdf`
